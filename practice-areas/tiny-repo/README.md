# Tiny Repo Practice Area

## Goal

Use a very small fake codebase to test exploration prompts, review prompts, and workflow wording.

## Files

- `app.py`: tiny script with a simple behavior
- `notes.md`: a few rough product notes

## Suggested Tasks

1. Ask Claude Code to summarize the repo.
2. Ask Claude Code to suggest one safe improvement.
3. Ask Claude Code to draft a small plan before editing.

## Reflection

Keep notes here about which prompts felt clear and which instructions were too vague.
